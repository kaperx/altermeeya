# AlterMeeya

A MangaMeeya launcher which fix the file path problem.
## Usage

1. Run altermeeya.exe as administrator. It will register a shell menu item.
2. Right click on the manga archive in explorer, and click "AlterMeeya" in the popup menu. For the first time, you will be asked to choose the path to "MangaMeeya.exe".

## Requirements

The manga archive file should be placed in a disk driver of NTFS format, because this program depends on the hard link feature of NTFS file system.


# AlterMeeya

本程序是一个MangaMeeya启动器，用于解决其无法处理带有特殊字符的文件路径的问题。

## 用法

1. 用管理员权限运行本程序。它会自动注册一个文件浏览器的右键菜单项。
2. 在文件管理器里，右键点击漫画文件包，在右键菜单中选择"AlterMeeya"即可。第一次打开的时候，AlterMeeya会让你选择"MangaMeeya.exe"的路径。

## 要求

漫画文件包必须放置在NTFS格式的磁盘分区下，因为本程序依赖于NTFS文件系统的hard link功能。
